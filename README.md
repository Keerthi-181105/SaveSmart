SaveSmart

SaveSmart is a modern web application that aggregates money-saving schemes and investment options from major Indian banks, making it easy to discover and compare financial products in one place.

Features

- Multi-Bank Coverage: Access schemes from 7 major Indian banks
- Modern UI/UX: Glassmorphism design with smooth animations
- Dark/Light Theme: Toggle between themes with persistent preferences
- Fully Responsive: Works seamlessly on desktop, tablet, and mobile
- Fast & Lightweight: Pure HTML, CSS, and vanilla JavaScript - no frameworks
- Easy Navigation: Quick access to all bank schemes and investment options
- Direct Links: All schemes link directly to official bank pages

Supported Banks

1. State Bank of India (SBI) - 7 schemes
2. HDFC Bank - 9 schemes
3. ICICI Bank - 9 schemes
4. Axis Bank - 3 schemes
5. Bank of Baroda (BoB) - 11 schemes
6. Canara Bank - 11 schemes
7. Indian Overseas Bank (IOB) - 14 schemes
8. Yes Bank - 3 schemes

Quick Start

Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No server required - runs entirely in the browser!

Installation

1. Clone the repository
git clone https://github.com/Keerthi-181105/SaveSmart.git
cd SaveSmart

2. Open in browser
Simply open index.html in your browser or use a local server (optional):

Python 3:
python -m http.server 8000

Node.js (with http-server):
npx http-server

3. Start exploring!

Design Features

Visual Elements
- Animated gradient background with floating orbs
- Glassmorphism effects for modern, frosted glass appearance
- Smooth scroll animations with intersection observer
- Hover effects with scale and color transitions
- Responsive grid layout adapting to all screen sizes

Color Scheme
- Dark Theme: Deep blue background with purple/cyan accents
- Light Theme: Soft white background with subtle shadows
- Brand Colors: Purple, Cyan, Accent Green

Usage

Navigate Banks
1. Open index.html in your browser
2. Browse through bank cards on the homepage
3. Click on any bank card to view its schemes

View Schemes
1. Each bank page lists all available schemes
2. Click on any scheme link to visit the official bank page
3. Use the "Back to all banks" button to return to homepage

Toggle Theme
- Click the Theme button in the header
- Your preference is saved automatically

Technologies Used

- HTML5: Semantic markup
- CSS3: Custom properties, Grid, Flexbox, Animations
- JavaScript (ES6+): Modern vanilla JS, no dependencies
- Google Fonts: Inter font family
